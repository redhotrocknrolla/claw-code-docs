## Зоны неопределённости

- Поведение инструментов в контексте конкретного MCP-хоста зависит от внешнего контекста.
- Типы `T` и `E` в обобщённых функциях (`fn tool_error`, `fn tool_success`) требуют уточнения по соседним модулям.
- Ограничения по сериализации (`serde_json`) следуют из зависимости, не из кода `lib.rs` напрямую.
